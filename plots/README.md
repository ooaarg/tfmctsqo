# Paper figures & numbers

Scripts that recompute every headline number and regenerate the benchmark
figures directly from `../runs/` (MCTS-Extreme vs DPSize vs GEQO).

## Run

```bash
uv run --no-project --with numpy --with pandas --with matplotlib --with scipy \
    python gen_results.py
uv run --no-project --with numpy --with pandas --with scipy \
    python gen_stats.py
```

`figures/` ships with the committed reference output. Figures sourced from the
baseline tracks (`aj-training`, `hq_training*`) and `mcts_process` are static and
not regenerated by these two scripts.
